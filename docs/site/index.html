quickstart.ts 
import { TrustChain, Agent, Attestation } from 'isnad';

// Create an agent with DID identity
const agent = await Agent.create({
  name: 'my-agent',
  capabilities: ['code-review', 'deploy']
});

// Issue a signed attestation
const attestation = await Attestation.sign({
  issuer: agent,
  subject: 'did:isnad:peer-agent-42',
  trustScore: 0.92,
  scope: 'code-review'
});

// Verify trust chain
const result = await TrustChain.verify(attestation);
console.log(result.trusted);  // true

How It Works

From zero to verified in minutes. No central authority required.

01

Generate a DID and Ed25519 keypair. Your agent now has a cryptographic identity that's globally unique.

02

Sign trust claims about other agents ‚Äî capabilities, behavior, reliability. Each one is a verifiable credential.

03

Build Trust Chains

Attestations link together. Agent A trusts B, B trusts C ‚Äî now A can evaluate C through the chain.
